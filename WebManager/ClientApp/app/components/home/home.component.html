WebManager 1.0 by PK

Welcome to WebManager, a brand new single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Webpack for building and bundling client-side resources
  • Bootstrap for layout and styling

Basic funcionalities planned be provided in final WebManager version

  • User authentication. Create new user and log in to view your personalised tasks list.
  • New tasks creation. Add new tasks to list to keep track.
  • Displaying current tasks. Monitor your current tasks.
  • Tasks modification. Modify existing ones to be always up-to-date.
  • Multiple users support. Share tasks with your friends and complete them as a team!
